📌 Project Overview

This project showcases an interactive Power BI dashboard designed as an Activity Tracking and Monitoring System for Civil Society Organizations (CSOs).

The dashboard enables stakeholders to track activities, monitor progress, and evaluate performance across multiple regions and reporting periods in a centralized and data-driven manner.

🎯 Objectives
Track CSO activities and implementation status
Monitor progress across locations and time periods
Identify delays, gaps, and performance trends
Support data-driven decision-making and reporting

🛠️ Tools & Technologies
Power BI – Dashboard design and visualization
SQL – Data extraction and transformation
Excel / CSV – Data preparation
DAX – KPI calculations and measures

📊 Key Features
✅ Activity tracking by organization, district, and time period
✅ KPI indicators for progress monitoring
✅ Interactive filters for dynamic data exploration
✅ Trend analysis of activity completion
✅ Comparative performance analysis across regions
